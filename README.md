# Popovers

[![Build Status](https://github.com/Pavel19740404/ahj-popovers/actions/workflows/deploy.yml/badge.svg)](https://github.com/Pavel19740404/ahj-popovers/actions/workflows/deploy.yml)

🔗 **[GitHub Pages — Live Demo](https://Pavel19740404.github.io/ahj-popovers/)**

Реализация Bootstrap-подобного Popover-виджета на чистом JavaScript без jQuery.

## Описание

Виджет отображается **сверху** кнопки и центрируется **горизонтально** относительно неё. Позиционирование в **px** (без translate/translate3d). Повторный клик скрывает popover, клик вне — тоже.

## Запуск

```bash
npm install
npm start      # дев-сервер
npm test       # тесты
npm run build  # сборка
```
